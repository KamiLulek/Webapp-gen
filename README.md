# Webapp-gen

download code

sudo cp webapp-gen /usr/local/bin/webapp-gen && sudo chmod +x /usr/local/bin/webapp-gen

run

  webapp-gen install
  webapp-gen -list
  webapp-gen -edit <nazwa>
  webapp-gen -remove <nazwa>

install files:

  Folder:   /usr/local/bin/webapp-list/App*
  Wrapper:  /usr/local/bin/webapp-list/App*/App*-app
  Cfg:      /usr/local/bin/webapp-list/App*/App*.cfg
  Desktop:  /home/user/.local/share/applications/App*.desktop
